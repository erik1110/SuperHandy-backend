## Getting Started

-   (1) Create the `config.env`. (You can get copy from `example.env`)
-   (2) Run the npm install.
-   (3) Depends on your environment and follow the script below.
-   (4) You can get [swagger-doc](http://localhost:3000/api-doc/).

```
// development
npm run dev

// production
npm run prod

// swagger docs
npm run swagger

```
